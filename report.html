   Review

1. What is a crypto scam?

2. What types of crypto scams exist?

Crypto scam - it’s a criminal deception intended to result in a crypto financial gain. [1]. Where are a big amount of scams, so it's better to divide it in groups:

Social - using social engineering to trick or manipulate victims

Software - use vulnerabilities in software to directly steal money

Governance - break the contract with customers for personal financial gain

Dirty coins - originated/involved in crime activity, such as narcotrafficking, money laundering, stolen, etc.. can be frozen any time.

     Mastering data

I have created a JSON schema for some of the sources (but not all), which can be found in the json-schema folder. In addition, I have developed a microservice prototype for data collection, consisting of the following components:

  1. Scraper - this component is responsible for scraping data from various sources. To accomplish this, I have written a scheduled scraping script (located in the "scraper" folder) using Python&Scrapy, as it is a simple and straightforward solution.
  2. API - the API is used to retrieve normalized data from different sources using Java/Spring (located in the "api" folder) , which is my area of expertise with 2-3 years of experience.
  3. Logging - I have utilized Graylog to track and address any issues that arise with the API&Crawler.
  4. Deploy - docker-compose deploy script

To test service - execute docker-compose.yml, import&execute postman query from
SCAM.postman_collection.json file.
